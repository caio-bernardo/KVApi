# KVApi

A Key-Value Store Web API project.

## Features

- [x] In Memory Database
- [x] CRUD Operations

## Usage

Clone this repo, go to the working directory and run de server.

```
cargo run
```

Make your requests

```
curl http://localhost:8080/health
```
